## 💡 Quick Examples

### Before (Hardcoded)
```javascript
const Page = () => {
    return <h1>"Welcome"</h1>;
}
```

### After (Internationalized)
```javascript
import { useTranslation } from '../../utils';

const Page = () => {
    const t = useTranslation("pages.home");
    return <h1>{t("welcome")}</h1>;
}
```

### Translation Files (Automatic)
```json
// en.json
{ "pages": { "home": { "welcome": "Welcome" } } }

// fr.json
{ "pages": { "home": { "welcome": "Bienvenue" } } }
```

## 🎓 Understanding Key Concepts

### Namespace
The translation scope based on your file path.
```
Component Path              → Namespace
src/pages/destination/...   → pages.destination
src/components/gallery/...  → components.gallery
src/dialogs/Confirm.js      → dialogs
```

### Translation Key
A unique identifier for each label.
```
Label (any language)        → Key
"No year available"         → noYearAvailable
"Aucune année disponible"   → noYearAvailable (same!)
```

### Hook Usage
How you use translations in your component.
```javascript
const t = useTranslation("pages.dates");
return <p>{t("noYearAvailable")}</p>;
```
